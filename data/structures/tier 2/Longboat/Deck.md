# Longboat Deck

## Features

### Fatal Breach - Passive

When the Deck is Destroyed, the Longboat is destroyed.

### Secure Cargo - Action

A PC within the Deck can secure loose cargo before it crashes across the ship. They make a Strength Roll. On a success, advance the Longboat’s countdown. On a critical success, advance the countdown twice.
